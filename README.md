### Основная концепция
Бот предназначен для организации подготовки студентов к экзаменационной сессии, предоставляя инструменты для планирования, тестирования и отслеживания прогресса по учебной дисциплине "Компьютерные сети".

### Функциональные модули
1. База тестовых заданий
    - Тестовые материалы для 3 курсов (для 56 и 57 групп)
    - Различные форматы вопросов: одиночный выбор, множественный выбор.
2. Система планирования
    - Генерация индивидуального плана подготовки на основе:
        - Текущей даты и даты экзамена
        (для улучшения в бота в будущем можно добавить: - Количества дисциплин для подготовки)
    - Учет календарных особенностей:
        - Выходных дней
        - Праздничных периодов
        - Распределения нагрузки по дням недели
3. Система учета подготовки
    - Отметка выполненных разделов плана
    - Статистика выполненных заданий
    (для улучшения в бота в будущем можно добавить: - Визуализация прогресса по каждой дисциплине)
4. Система напоминаний
    - Настраиваемая периодичность уведомлений
    - Автоматическое увеличение частоты напоминаний по мере приближения сессии
    - Возможность отключения уведомлений
5. Аналитический модуль
    - Персональная статистика подготовки
    - Агрегированная статистика по группам пользователей
    - Еженедельные отчеты о прогрессе
6. Административный интерфейс
    - Форма обратной связи с администрацией
    - Система предложений по улучшению

### Технические особенности
- Реализация на языке Go без использования сторонних библиотек
- Модульная архитектура для легкого расширения функциональности
- Система хранения данных в структурированных файлах

### Целевая аудитория
Студенты групп 56 и 57.
